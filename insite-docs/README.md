# FrameForge for Dummies - Docs Index

Welcome. This in-app guide is written for first-time users. It explains what each tab does, what the options mean, and how to finish a run from upload to download.

## Read this first
- FrameForge is a local tool for preparing image datasets and (optionally) training LoRA models.
- You do not need to know AI terms to use the basic flow. Just follow the steps.
- This guide does NOT cover installation. It only explains how to use the app.

## Glossary (plain English)
- Run: One job you launch from a ZIP upload.
- Run ID: A 6-digit number used to track a run.
- Dataset: The final folder of images + text tags.
- Tags: Comma-separated labels saved per image.
- LoRA: A small training output you can load in an image model.
- Preset: A saved set of rules or settings.
- Queue: The list of runs waiting or working.
